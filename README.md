프로그램 소개

간단한 TPS 액션 게임입니다.
적들을 처치하고 동전을 모아 상자에서 좋은 성능의 무기를 획득하세요!
적들의 공격에 맞아 체력이 0이 되면 패배하고, 패배하지 않고 보스를 무찌르면 승리합니다.

조작

이동 : WASD

점프 : SPACE

구르기(고속이동) : Lshift (쿨타임 5초)

회복(25) : Q (쿨타임 5초)
